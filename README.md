# **⚙️ Hub Pro \- Edição Master**

Uma suite profissional e completa de ferramentas técnicas, cálculos geométricos avançados e conversores de precisão, tudo numa interface moderna e otimizada para dispositivos móveis.

## **🌟 Destaques da Versão Master**

O **Hub Pro** foi desenvolvido para profissionais e estudantes que necessitam de rapidez e precisão. Esta versão utiliza a paleta visual **Amber Gold**, focada em legibilidade de alto contraste e uma estética executiva.

### **📐 Geometria e Perímetros (Total de 21 Modelos)**

* **Perímetros Planos:** Cálculo de contorno para 7 figuras (Quadrado, Retângulo, Triângulo, Círculo, Trapézio, Losango e Elipse).  
* **Áreas (2D):** Fórmulas completas para superfícies planas.  
* **Volumes (3D):** Cálculos espaciais para Cubos, Cilindros, Esferas, Pirâmides e muito mais.  
* **Fórmulas Integradas:** Cada cabeçalho exibe a fórmula matemática utilizada e dicas de medição.

### **💱 Conversores Ultra-Completos**

* **Moedas:** Taxas para BRL, USD, EUR, GBP, JPY, entre outras.  
* **Técnicos:** Temperatura, Comprimento, Massa, Velocidade, Volume e Área.  
* **Avançados:** Dados Digitais (Bytes a Terabytes) e Pressão (Bar, PSI, ATM).

### **🛠️ Utilidades de Negócio**

* **Gestão de Combustível:** Cálculo de logística e custos de viagem.  
* **Finanças Rápidas:** Simulador de descontos e acréscimos percentuais.

## **📱 Design & Experiência do Utilizador (UX)**

* **Mobile-First:** Navegação em "pills" horizontal para ecrãs pequenos.  
* **Modo Escuro (Dark Mode):** Alternância instantânea com preservação de contraste.  
* **Amber Gold Palette:** Design sofisticado com sombreados suaves e texto em Slate escuro para evitar fadiga ocular.  
* **Sistema de Cópia:** Resultados prontos para partilhar com um clique.

## **🚀 Como instalar no GitHub Pages**

Este projeto é uma aplicação **Single-File** (HTML, CSS e JS num único ficheiro), o que facilita imenso o alojamento.

1. Crie um novo repositório no GitHub.  
2. Carregue o ficheiro HubPro.html e mude o nome para index.html.  
3. Vá a **Settings** (Definições) \> **Pages**.  
4. Em **Build and deployment**, selecione o branch main.  
5. Aceda ao link gerado pelo GitHub para começar a usar.

## **🛠️ Tecnologias Utilizadas**

* **HTML5 & CSS3:** Variáveis de cor nativas e Flexbox/Grid para total responsividade.  
* **JavaScript (Vanilla):** Motor de cálculo puro, sem dependências externas, garantindo carregamento instantâneo.  
* **Google Fonts:** Tipografia 'Inter' para máxima legibilidade técnica.

Desenvolvido com foco na precisão e elegância visual.
